# Hospital Patient Analytics (SQL Project)

## Project Overview
This project analyzes hospital operations using SQL to uncover insights related to:
- patient demographics and disease trends
- doctor workload and department activity
- appointment behavior
- billing and revenue patterns

The goal is to simulate how a data analyst supports healthcare decision-making using structured data.

---

## Dataset Description
This project uses six relational datasets:

- **patients** ‚Äì patient demographics, disease, admission details  
- **doctors** ‚Äì doctor specialization and department assignment  
- **departments** ‚Äì hospital department information  
- **appointments** ‚Äì appointment scheduling and status  
- **billing** ‚Äì payment amounts and methods  
- **medical_records** ‚Äì diagnosis, treatment, and cost details  

---

## SQL Skills Demonstrated
- SELECT, WHERE, ORDER BY
- INNER JOIN, LEFT JOIN
- GROUP BY, HAVING
- Aggregation functions (SUM, AVG, COUNT)
- Window functions (DENSE_RANK)
- Subqueries and real-world analytics logic

---

## Key Insights (Findings)
- Identified the most common diseases treated by the hospital
- Analyzed appointment volume across departments
- Found top revenue-generating payment methods
- Measured average treatment costs by diagnosis
- Highlighted departments with the highest patient workload
